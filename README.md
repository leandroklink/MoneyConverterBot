# Bot de Conversão de Moedas - Telegram

Bot simples desenvolvido em Python que realiza conversão de moedas utilizando a API da AwesomeAPI.

## Funcionalidades
- Escolha da moeda de origem
- Escolha da moeda de destino
- Conversão de valores em tempo real
- Integração com a API de câmbio
- Interface via Telegram

## Tecnologias utilizadas
- Python
- TeleBot (pyTelegramBotAPI)
- Requests
- AwesomeAPI (cotações)

## Como funciona
1. O usuário inicia o bot com /start
2. Escolhe a moeda de origem
3. Escolhe a moeda de destino
4. Informa o valor para conversão
5. O bot retorna o valor convertido

## Observações
- A chave do bot do Telegram não está incluída por segurança
- É necessário criar seu próprio bot com o BotFather
- O ambiente virtual venv não está versionado

## Executar o projeto

pip install pyTelegramBotAPI requests
python RealConverter.py
